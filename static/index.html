devRant Community Project 2016

The tracking website for the devRant stressball in a bottle project

The STOTV (Stressball Trans-Oceanic Trackable Voyage) is a devRant sponsored community project that takes the idea of a message in a bottle into the 21st century. This site is the web portal to display the location of our 21st century message in a bottle.
