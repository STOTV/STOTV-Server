Current API Endpoints

/api/v1/send [POST]

used only by RockBLOCK servers

/api/v1/found/[imei]

Returns recovery information for a device. In json format with fields deviceName, returnName, returnAddr1, and returnAddr2

/api/v1/location or /api/v1/location/[id]

Returns last 20 pings for registered device(s). In json array format with fields name, time, latitude, longitude, and cep
